粘贴板配置 ​

介绍 ​

YinTao 作为底层引擎，经常会在多个平台间使用，为了让用户能够更好的使用YinTao功能，以及兼容一些平台实现的差异。 我们提供了通用的粘贴板配置模式。

原理 ​

在执行中或录制中，平台将传递一系列参数来控制具体执行的行为，现在我们允许用户去覆盖或扩充它 (注: 仅针对本地有效) 。

我们读取用户的粘贴板内容，转化为配置文件，并覆盖原始配置(仅针对修改项)。

让用户可以更加灵活的时候扩展功能。

示例 ​

使用方式: 复制下列代码至粘贴板

自定义录制打开的浏览器 ​

json
{
    "url": "https://jd.com"
}

json
{
    "url": "https://jd.com",
    "device":"iPhone 6"
}
